### 4Ô∏è‚É£ VERIFY WITH GOOGLE (After Launch)

Once your site is live:

1. **Google Search Console**
   - Go to: https://search.google.com/search-console
   - Add your property (website)
   - Copy verification code
   - Add to `/app/layout.tsx` line 84:
     ```typescript
     verification: {
       google: 'paste-your-code-here',
     }
     ```

2. **Submit Sitemap**
   - In Google Search Console
   - Go to "Sitemaps"
   - Submit: `https://yoursite.com/sitemap.xml`

3. **Test Your Pages**
   - Rich Results Test: https://search.google.com/test/rich-results
   - Test each page URL to verify structured data

## üÜò TROUBLESHOOTING

**Q: Social share image not showing?**
- Clear Facebook cache with Sharing Debugger
- Verify image is exactly 1200x630px
- Ensure filename is `/public/images/og-image.jpg`

**Q: Pages not in Google?**
- Submit sitemap in Search Console
- Check robots.txt isn't blocking
- Wait 1-2 weeks for indexing

**Q: Structured data not working?**
- Test with Rich Results Test
- Verify no JavaScript errors in console
- Check schema syntax in page source
