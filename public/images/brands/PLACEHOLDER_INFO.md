# Brand Logo Placeholders

The following PNG logo files need to be added to this directory:

## Required Logo Files:

1. **gs-education.png** - GS Education logo
2. **saco.png** - SACO logo  
3. **wings.png** - Wings logo
4. **yoi-scissors.png** - Yoi Scissors logo
5. **ibiza-brushes.png** - Ibiza Brushes logo

## Existing Logo Files:
- ✅ loreal.png (L'Oréal Professionnel)

## Specifications:
- **Format**: PNG with transparent background
- **Recommended size**: 400px wide x 120px tall
- **Color**: Grayscale or brand colors (will be displayed in grayscale with color on hover)
- **File naming**: Use lowercase with hyphens (e.g., gs-education.png)

## Where logos are displayed:
1. **Homepage**: Under the hero section in a horizontal strip
2. **About page**: In the "Trusted By Industry Leading Brands" section

## Temporary Fallback:
Until real logo PNG files are added, the brand names will display as text placeholders.
